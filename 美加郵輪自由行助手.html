當地文化顧問
必看!最容易踩雷的三件事 1. 小費一定要給(餐廳、計程車、行李員)
2. 價錢標籤都不含稅(結帳會多 12% 左右)
3. 過馬路一定要走斑馬線並按紅綠燈鈕(罰款很重)
小費怎麼給? ▶

餐廳坐下來吃飯:帳單金額的 15%~20%(稅後不算,看小計)

計程車 / Uber:車資的 10%~15%

飯店行李員:每件行李 1~2 加幣 / 美元

飯店打掃員:每晚 2~5 加幣放在床頭

郵輪上:大多已自動扣每日服務費(約 USD 16~18/人/天),不必另給。但若要客房服務或請船員幫忙,可給 1~2 美元小費

免小費:速食店(麥當勞)、自助式咖啡店、買東西、藥房

實用秘訣 刷卡結帳時,機器會問 Tip(小費),選 15% 或 18% 即可。不要按 No Tip 按鈕,服務生會很尷尬。
稅金 / 標價問題 ▶

美加的標價 不含稅,結帳時加上稅金:

• 溫哥華 (BC省):5% GST + 7% PST = 12%

• 餐廳另加 7% PST 賣酒精;一般食物只收 5% GST

• 阿拉斯加郵輪在美國港口:各州不一,大多 5%~10%

• 看到 $10 衣服,實際付約 $11.20,加小費就更多

建議 身上多備一些 1元、5元紙鈔方便給小費。可以在機場或飯店換零錢。
過馬路 / 走路規矩 ▶

• 絕對要走斑馬線!不要為了抄近路穿越馬路(叫 jaywalking,會被罰款,溫哥華有警察取締)

• 大多紅綠燈要 按一下牆上的按鈕,小綠人才會亮

• 看到「WALK」/小綠人 → 可以過;看到「DON'T WALK」/紅手 → 等

• 倒數計時是「給已經在過的人走完」, 不要在倒數時開始過

• 在郊區或步道上,人行道沒裝路燈,夜間不要走

廁所 / 飲水 ▶

• 廁所叫 Restroom 或 Washroom(加拿大常用)

• 路上很少免費廁所!出門前先用,在景點、商場、星巴克使用

• 星巴克、麥當勞:消費後可問店員密碼進入(密碼通常印在發票上)

• 自來水可以喝,溫哥華水質很好。可以帶水壺裝水,飯店、機場、商場都有飲水機

• 餐廳:坐下後服務生會問 "Water?",點頭就會免費供應冰水

餐廳用餐流程 ▶

1. 等帶位:進門後 站在「Please Wait to be Seated」牌子旁等服務生帶你入座,不要自己找位子

2. 點餐:同一個服務生會負責你整桌(他/她會自我介紹名字)

3. 加水:水杯空了,服務生會自動來加,免費

4. 結帳:說 "Check, please" 或舉手示意,帳單會送到桌上,不用走到櫃檯

5. 給小費:放在桌上(現金)或刷卡時選擇

份量很大! 美加餐點份量是台灣 1.5~2 倍,建議兩位長輩共點一份+加點湯/沙拉。吃不完可以說 "To go, please" 打包帶走。
郵輪船上禮節 ▶

• 每天會在房間放當日節目單 (Daily Program / Princess Patter / Compass) — 一定要看

• 船上時間 vs 港口時間有時不同,以船上時間為準

• 正式之夜 (Formal Night):晚餐穿正式一點,但不強制

• 主餐廳座位:固定時段(傳統用餐)或自由(任何時段),以登船時收到的卡為準

• 消費全靠船卡,船上不收現金,離船結算

• 下船觀光:準時回船!船不等人,郵輪離港時間是「船上時間」

非常重要 下船時記得 帶好船卡 + 證件影本。回船刷卡進入,沒帶就上不了船。手機照下護照首頁存起來備用。
與當地人互動 ▶

• 北美人 愛打招呼:進電梯、店員會說 "Hi, how are you?" → 回 "Good, thank you" 即可

• 排隊文化嚴格,絕對不要插隊,即使只差一個人

• 開門時若後面有人,會 幫忙扶住門,記得說 "Thank you"

• 人和人講話 距離較遠(約一個手臂),不要太靠近

• 不要大聲講電話,尤其餐廳、火車、巴士

• 跟陌生人 不問年齡、薪水、婚姻狀況

溫哥華 / 加拿大 特殊事項 ▶

• 溫哥華華人非常多(列治文/Richmond 區尤其多),很多店有中文服務

• 大麻已合法(19歲以上),路上有時會聞到味道,不必驚慌,別讓孫輩聞

• 加拿大用 加幣(CAD),美國用 美金(USD),郵輪船上以 美金計

• 加拿大電壓 110V,跟台灣一樣,插頭也一樣可直接用

• 5月~9月 溫哥華氣溫 15~25°C,早晚涼,洋蔥式穿搭

• 阿拉斯加冰河區更冷,需備外套、毛帽、手套(船上會發送毛毯)

人在郵輪上的緊急狀況 ▶

船上總機:房間電話按 0 或 911(船內專線,非報警)

船上保安:找最近的工作人員,說 "Security, please"

船上醫療:每艘船都有醫療中心(通常 Deck 4-5),24小時駐診

失蹤同伴:到 Guest Services 客服中心請廣播尋人

東西忘在港口岸上:船離港後幾乎拿不回,但可請 Guest Services 幫忙詢問港口辦公室

來不及上船! 若郵輪已開,立刻聯絡郵輪公司港口代理 (Port Agent — 名單在每日節目單最後一頁),他們會協助安排你坐車或飛機到下一個港口會合。
絕對不要自己決定放棄繼續行程,先打給 Port Agent。
迷路了怎麼辦? ▶

1. 不要慌,找安全的地方坐下(咖啡店、書店、商場),不要在街角徘徊

2. 開手機 Google Maps → 看自己藍點 → 看飯店在哪 → 點「路線」走回

3. 找警察、店員、旅館前台問路(可出示翻譯助手裡的卡片)

4. 叫 Uber / Lyft 是最簡單的方法 — 只需要在 App 輸入飯店地址,司機就會來接

5. 若沒網路:路邊找有 Wifi 的咖啡廳(星巴克、Tim Hortons)連上後再查

事先準備 出門前在 Google Maps 上 下載溫哥華離線地圖 → 沒網路也能用,長輩比較不用擔心漫遊費。
常見詐騙 / 危險地帶 ▶

溫哥華要避開的區域:

• Hastings & Main 一帶 (Downtown Eastside) — 街友、毒品問題嚴重,白天也不要去,跟唐人街(Chinatown)只隔一條街,要小心

常見伎倆:

• 假裝問路或要簽名請願,趁機扒走錢包

• ATM 旁有人「好心教學」,記下密碼

• 街頭問你能不能換零錢,趁機掉包

防範:

• 錢包放胸前包包,不要放後口袋

• 大鈔不要露白,結帳前先把錢分好

• 有人接近就大聲說 "No" 並走開

天然災害 (地震/海嘯) ▶

地震:太平洋西北岸偶有地震,但多為小震。若搖晃 → 「趴下、掩護、穩住」(Drop, Cover, Hold On)。躲到桌下,遠離窗戶。

海嘯:港口或郵輪上有警報,聽從指示往高處走。

暴雨/濃霧:常見於溫哥華,郵輪可能改變航線(冰河灣天氣不好就會跳過),不要失望,船上活動很多。

慢性病用藥 — 出發前準備 ▶

1. 帶足全行程的藥 + 多帶 5-7 天備用(萬一行程延誤)

2. 藥裝原始藥盒裡,標籤完整(海關抽查時可證明)

3. 請醫生開英文處方箋(英文藥名 + 劑量),一份隨身,一份放行李

4. 隨身行李帶藥,不要全放託運(行李遺失就糟了)

5. 製作中英文用藥清單卡(見上面病歷卡)

常見慢性病藥英文名 高血壓 → Antihypertensive (Amlodipine 脈優, Losartan 可悅您)
糖尿病 → Antidiabetic (Metformin 庫魯化錠)
膽固醇 → Statin (Atorvastatin 立普妥, Crestor 冠脂妥)
心臟 → Beta-blocker (Bisoprolol)
抗凝血 → Anticoagulant (Warfarin 可邁丁, Aspirin 阿斯匹靈)

什麼狀況該就醫? ▶

立刻 911 / 急診:

• 胸痛、呼吸困難、心律不整

• 突然頭暈、口齒不清、半邊無力(中風徵兆)

• 跌倒撞到頭、骨折、大量出血

• 高燒不退超過 39°C

• 嚴重過敏(蕁麻疹+喉嚨腫)

看門診 (Walk-in Clinic):

• 一般感冒、輕微腸胃不適、皮膚問題

• 慢性藥用完了需要再開

找藥房諮詢:

• 輕微頭痛、便秘、腹瀉、失眠、過敏

郵輪上看醫生 ▶

• 每艘船有 Medical Center(醫療中心),通常 Deck 4 或 Deck 5

• 開放時段每天 8-10am, 4-6pm 左右,其他時段急診打 911(船內專線)

• 看診費約 USD 100-300,藥物另計,先付費再申請保險

• 暈船很常見:可在郵輪 Guest Services 索取 免費暈船藥(粉紅色 Bonine)

• 嚴重狀況時船醫可決定就近港口下船送醫

暈船預防 • 上船第一天先到 Guest Services 拿一條暈船貼或藥
• 房間選 船中段、低樓層較不晃
• 多到甲板透氣、看遠方
• 不要空腹也不要太飽

走失了該怎麼辦?(請冷靜,按順序做)
第 1 步:停下,別亂走 ▶

• 站在原地 30 秒,環顧四周,可能家人就在附近

• 如果還在剛才的店家、景點裡,先繞一圈再離開

• 不要因為慌張往奇怪方向跑

第 2 步:找安全的地方坐下 ▶

• 找 有店員的地方:咖啡店(星巴克、Tim Hortons)、便利商店、藥房、書店

• 不要在街角站太久(怕不安全)

• 坐下後深呼吸,拿出手機

第 3 步:用本助手分享位置 ▶

• 回到本頁(找回家人),按 「取得我目前位置」

• 允許瀏覽器使用定位

• 按 「分享給家人」 → 選 LINE / 簡訊 / Email,把位置傳出去

• 或按 「複製連結」,自己貼到 LINE

第 4 步:撥打家人電話 ▶

• 在下方「家人聯絡」直接點擊撥打

• 告訴家人:「我走失了,我現在在 ___(店名/路名),我會在這裡等你」

• 把位置傳出去後,留在原地等

第 5 步:若 30 分鐘內找不到 → 前往集合點 ▶

• 看下方「預設集合點」,前往最近的

• 主要集合點:Aercorn Hotel 飯店大廳(在溫哥華時)

• 若在郵輪上:Guest Services 客服中心 Deck 4

• 若在港口:郵輪碼頭/Tender 接駁碼頭

第 6 步:緊急狀況才打 911 ▶

★ 出發前要做的 6 件事(很重要)
1. 約定每日「三次集合」 ▶

例如:

• 早餐 08:00(自助餐廳 Windjammer)

• 晚餐 18:00(主餐廳 Reflections)

• 晚上 22:00 回房間

沒按時出現就知道有狀況,可以開始找

2. 互換船卡照片 ▶

• 每個人船卡(SeaPass)上有姓名 + 房號

• 全家互拍對方的船卡照片存手機,萬一掉了能告知 Guest Services

3. 把重要資訊寫小紙片放錢包 ▶

用一張小紙片(信用卡大小),用大字寫上:

• 自己姓名(中英文)

• 飯店名:Aercorn Hotel, Vancouver

• 郵輪船名:Serenade of the Seas + 自己房號

• 家人台灣電話 + 在台灣聯絡人姓名

• 「If lost, please help me call this number 如果走失,請幫我打這個電話」

4. 設定 LINE 即時位置分享 ▶

LINE 內建「分享位置」功能(會自動更新位置 1 小時):

• 開啟與家人的 LINE 聊天

• 點 + 號 → 位置 → 分享我的位置

• 選「分享 1 小時」,家人可以看到你即時移動

• 下船時就開啟,回船時關掉

5. 啟用「Find My」/「Find My Device」 ▶

iPhone: 設定 → 自己的名字 → 尋找 → 尋找我的 iPhone(打開)

Android: 設定 → Google → 尋找我的裝置(打開)

在台灣的家人即使沒接到電話,也能透過 iCloud / Google 帳號看到長輩手機位置

★ 把長輩的 Apple ID / Google 帳號密碼寫下給家人

6. 預先下載離線地圖 ▶

沒有網路時還能用:

• Google Maps → 右上頭像 → 離線地圖 → 選擇地圖 → 框出溫哥華 + 阿拉斯加各港口下載

• 每個港口下船前再開一次離線地圖確認可用
